## 11321 - Sort! Sort!! and Sort!!!

#### Question Meaning
直到輸入 0 0 為止，輸入m（排序的數值數量）以及n（mod n）。輸出需按照規則排列：
- mod n 的值小的在前面。
- 偶數在前面。
- 若是偶數，數字由大排到小。
- 若是奇數，數字由小排到大。

#### Notice
1. 數值中可能有負數，需考慮進去。（特別注意判斷奇偶的地方）
2. 第一行要記得輸出m和n，特別容易忘記最後的0 0也要輸出！！
3. 不能爆開，會TLE。

#### Note
- ascending：上升。
- modulo：模數、取模（mod）。
- precede：優於。
- remainder：餘數。
- denotes：表示。
